## Working with mamp

uncomment in php.ini

```
extension=intl
extension=mysqli
extension=pdo_mysql
```

---

## commands

create new project
start server
stop server

```
symfony new --webapp my_project
symfony server:start
symfony server:stop
```

create database

```
symfony console doctrine:database:create / symfony console do:da:cr
```

make controller

```
symfony console ma:con home
```
